## 1. 测试策略

本项目核心是**传输无关三层架构**（[Adapter](../../adapters/mock_cli.py) ↔ [Router](../../core/router.py) ↔ [Handler](../../core/handler.py)），测试也分层进行：

| 测试类型 | 方法 | 目的 | 环境 | 分册 |
|---------|------|------|------|------|
| **离线单测** | 注入假 `transport`/embedder/fetcher、临时 db、localhost | 验证 LLM 消息组装、知识库(含 schema 校验/克数规格)、RAG、结构化商品库、微信客服/智能机器人适配器、人工接管与转人工双通道、SQLite 持久化、REST API、Web 工作台、可观测计数、端到端集成、并发安全、路由链路、入口装配与启动自检（现有 16 套，含内容完整性护栏） | 任意平台，无需外网/密钥 | [01-离线单测](01-离线单测.md) |
| **LLM 评测** | `examples/demo_agent.py` 真调模型 | 验证私域维护场景回复质量（走知识库/引导下单/转人工/防注入/多轮记忆） | 需配好 `.env` 密钥 | [02-评测集](02-评测集.md) |
| **真机手测** | `MOCK=1` 走 [MockCliAdapter](../../adapters/mock_cli.py) | 验证全链路（去重/限流/触发/session/回复） | 本地即可（Mac 可用） | [03-真机与集成测试](03-真机与集成测试.md) |
| **断网降级** | 拔网或改错 base URL | 验证 HTTP错/超时/JSON错/空响应时返回兜底、不抛异常 | 任意平台 | [03-真机与集成测试](03-真机与集成测试.md) |

当前已有 **16 套离线测试**（`tests/test_*_offline.py`），全部纯离线（注入假 `transport`/embedder/fetcher、临时 db、localhost），无需外网/密钥，可在任意平台（含 Apple Silicon Mac）直接运行。另有 **evals 金标评测**（`evals/run_evals.py`：检索层离线可回归，`--online` 评真实回复层与合规红线）。真实企微 hook（`ntwork`）仍待 Windows 环境；微信客服适配器已实现（离线测试已覆盖），真机公网回调联调待验证（操作手册见 [技术文档 07-真机联调SOP](../技术文档/07-真机联调SOP.md)）。

## 2. 快速运行

```bash
# 一键全跑（16 套离线测试，纯离线，无需外网/密钥）
for t in tests/test_*.py; do python3 "$t"; done

# 离线评测（检索层金标，无需外网/密钥；--online 加评真实回复层）
python3 evals/run_evals.py

# 单项运行
python3 tests/test_llm_offline.py           # LLM 消息组装、兜底、截断、角色映射、防注入
python3 tests/test_knowledge_offline.py     # 知识模块可插拔（Static/Rag/Structured/Hybrid）
python3 tests/test_rag_offline.py           # RAG 混合检索/小库直通/兜底/缓存
python3 tests/test_structured_kb_offline.py # 结构化商品库精确匹配 + Hybrid 组合
python3 tests/test_kf_offline.py            # 微信客服适配器：AES/签名/回调映射/send/降级
python3 tests/test_takeover_offline.py      # 人工接管 + 结构化转人工
python3 tests/test_sqlite_offline.py        # SQLite 持久化与重启恢复
python3 tests/test_api_offline.py           # REST API 端点与鉴权
python3 tests/test_integration_offline.py   # 端到端串 T1+T2+T3
python3 tests/test_concurrency_offline.py   # 会话存储并发安全（锁）
python3 tests/test_console_offline.py       # Web 工作台：登录/代理注入 token/CSRF/兜底
python3 examples/demo_agent.py              # 17 场景私域维护评测（真调 LLM，含合规对抗场景）
MOCK=1 HANDLER=echo MOCK_SCRIPT=examples/sample.json python3 main.py  # echo 管道回归
```

> REST API 实现见源码 [api_server.py](../../api_server.py)。
